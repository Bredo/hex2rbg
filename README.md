hex2rbg
=======
hex2rbg is problaly the smallest HEX/RBG-converter in JavaScript ever.
hex2rbg got two funktions: h2r() and r2h().

h2r(hexcolor)
-------------
Works by one input: the hex.
Examlpe:
alert(h2r("#FF0000"));

r2h(r, b, g)
------------
Works by three inputs: the rbg-values.
Examlpe:
alert(r2h(50, 50, 50));
